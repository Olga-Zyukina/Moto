# Online store MOTO
A multi-page website. Directory wp - WordPress layout.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Mixitup.js
- BaguetteBox.js
- List.js
- CMS Wordpress

## Live Demo

Check out the live demo: [MOTO](https://olga-zyukina.github.io/Moto/)

## Project Structure

```
├── css/         # Cascading Style Sheet files
├── img/         # Images
├── js/          # JavaScript files
├── source/      # Software libraries
└── wp/          # WordPress layout
```
